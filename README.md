# Belajar Machine Learning Bahasa Indonesia (Ongoing)
Ini adalah repository untuk belajar machine learning dalam bahasa Indonesia dan bersifat gratis.

Akan ada beberapa bahasan di tutorial Machine Learning ini, yaitu:

1. Linear Regression
2. Logistic Regression
3. Support Vector Regression
4. Support Vector Classification
5. Decision Tree Classification
6. Random Forest Classification
7. Naive Bayes (Classification)
8. K-Nearest Neighbors (Classification)
9. Gradient Boosting Regression
10. Gradient Boosting Classification
11. Simple Regression Artificial Neural Network (TensorFlow)
12. Simple Classification Artificial Neural Network (TensorFlow)
