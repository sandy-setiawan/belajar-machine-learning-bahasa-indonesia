# Belajar Machine Learning Bahasa Indonesia (Ongoing)
Ini adalah repository untuk belajar machine learning dalam bahasa Indonesia dan bersifat gratis.

Akan ada beberapa bahasan di tutorial Machine Learning ini, yaitu:

1. Linear Regression
2. Logistic Regression
3. Support Vector Regression
4. Support Vector Classifier
5. Decision Tree Classifier
6. Random Forest Classifier
6. Naive Bayes (Classification)
7. K-Nearest Neighbors
8. Gradient Boosting (Regression dan Classification)
9. Simple Artificial Neural Network (TensorFlow)
